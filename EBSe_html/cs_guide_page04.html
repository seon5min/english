ID/PW 찾기

회원가입 여부 확인 또는 사용중인 아이디와 비밀번호를 확인 하실 수 있습니다.
SNS계정인증 회원의 경우 가입한 SNS 사이트에서 확인 가능합니다.

ID(아이디) 찾기
  • - 가입여부 확인과, 가입 시 등록한 아이디를 확인하실 수 있습니다.
  • - 아이디를 찾기 위해 로그인 페이지로 이동하시어 등록된 회원정보를 입력하여 본인인증을 합니다.
아이디 확인
  • - 본인인증이 완료되면 가입하신 아이디를 확인하실 수 있습니다.
  • - 아이디 확인 후 로그인 버튼 클릭 시 로그인 페이지로 이동합니다.
  • - 아이디 확인 후 비밀번호 재발급 버튼 클릭 시 비밀번호 재발급 페이지로 이동합니다.
비밀번호 찾기
  • - 아이디를 찾기 위해 로그인 페이지로 이동하시어 등록된 회원정보를 입력하여 본인인증을 합니다.
  • - 비밀번호 재발급을 위해 등록된 회원정보를 입력하여 본인인증을 합니다.
비밀번호 재발급 방법 선택
  • - 본인인증이 완료되면 이메일로 발급, 등록하신 휴대폰으로 발급 중 한가지를 선택하여 비밀번호를 재발급 받으실 수 있습니다.
비밀번호 재발급 방법 선택_휴대폰 인증
  • - 인증 받으실 휴대폰의 번호를 정확히 입력 후 인증번호 받기를 클릭 합니다. 휴대폰으로 전송된 인증번호를 입력하신 후 확인 버튼을 클릭 합니다.
비밀번호 변경
  • - 원하시는 비밀번호로 변경하실 수 있습니다.
