    EBS 영어교육채널 활용 설명회2

  • 2022.05.28 3,307

  • 2023년 교육박람회 이벤트 클래스 참가 신청

    1. 마스터 유진의 자녀 영어교육법 특강 2. 리즐쌤과 함께하는 영어로하는 종이접기 클래스

    2023 대한민국 교육박람회 EBS부스에서 2가지 특강이 진행됩니다.첫째, EBS 부동의 1위 영어 방송 EBS 왕초보영어의 마스터유진 선생님이 자녀 영어교육에 관심이 많으신 학부모님을 위해 특강을 준비하였습니다. 둘째, 리즐쌤과 함께하는 영어로하는 종이접기 클래스를 마련하였습니다. 많은 참여 부탁 드립니다.

    class 1

    • 클래스명 : 마스터 유진의 자녀 영어교육법 특강 “우리아이 영어 실력이 달라졌어요”
    • 클래스 일정 : 2023.01.14 (토) 14:30 ~ 15:20
    • 대상 및 인원 : 자녀 영어교육에 관심있는 부모 30명
    • 강사 : 마스터 유진
    • 장소 : 2023교육박람회 코엑스 EBS 부스

    class 2

    • 클래스명 : 리즐쌤과 함께하는 영어로하는 종이접기 클래스
    • 클래스 일정 : [1차] 2023.01.12 (목) 14:00 ~ 14:50 [2차] 2023.01.13 (금) 14:00 ~ 14:50
    • 대상 및 인원 : [1차] 초등 1~3학년 학생 10명 [2차] 초등 4~6학년 학생 10명
    • 강사 : 리즐 선생님
    • 장소 : 2023교육박람회 코엑스 EBS 부스
