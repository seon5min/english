    네이티브는 쉬운 영어 표현으로 말한다! 왕초보 영어 탈출 특강

    2021 대한민국 교육 박랍회 EBS부스에서 영어 말하기에 자신감을 불어 넣어 줄 특강이 진행됩니다. 한국인이 많이 헷갈리는 영어 표현부터 영어 회화 잘하는 법, 우리 아이 영어 공부법 등의 알찬 내용들로 구성된 왕초보 영어 탈출 특강! 지금 바로 신청하세요!

    • 클래스 일정 : 2021년 5월 17일 14시
    • 대상 및 인원 : 학부모&성인 / 10명 추첨
    • 강사 : 마스터 유진 (강사는 변경될 수 있습니다.)
    • 준비물 : 왕초보 영어로부터 탈출하고 싶은 굳은 마음가짐

    ※ ‘이벤트 참여하기’ 버튼을 클릭하여 이벤트 참여 후 이벤트 클래스 신청정보를 작성하셔야 이벤트 클래스 당첨 대상이 됩니다.

    이벤트 참여 방법

    • step01 : 이벤트 참여 버튼을 클릭, 개인정보 이용에 대한 설명 확인 후 이벤트에 참여합니다.
    • step02 : 이벤트 참여 팝업창에서 이벤트 클래스 신청정보를 작성합니다.
    • step03 : 이벤트 종료 후 참가신청서를 작성하신 응모자를 추첨하여 이벤트 클래스에 참여할 기회를 드립니다!
