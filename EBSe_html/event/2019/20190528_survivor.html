    English 서바이벌 시청소감 남기기

    신규 방송프로그램 <English 서바이벌> 영상을 다시보기 후 시청소감을 작성해주세요.
    우수한 시청소감을 선정하여 푸짐한 경품을 드립니다.

    • 이벤트 기간 : 06월 02일 (일) ~ 06월 23일 (일)
    • 당첨자 발표 : 06월 28일 (금)

    ※ ‘이벤트 참여하기’ 버튼을 클릭하여 이벤트 참여 후 시청소감을 작성하셔야 상품 추첨 대상이 됩니다.

    이벤트 참여 방법

    • step01 : 이벤트 참여 버튼을 클릭, 개인정보 이용에 대한 설명 확인 후 이벤트에 참여한다.
    • step02 : <English 서바이벌> 프로그램을 선택하여 다시보기’ 시청 후 시청소감을 작성한다.
    • step03 : 이벤트 종료 후 우수한 시청소감을 작성하신 응모자를 추첨하여 푸짐한 경품을 드립니다.

    상품 안내

    방송프로그램 <English 서바이벌>의 우수한 시청소감을 작성해주신 응모자 중 총 100명을 추첨하여 파리바게뜨 블루베리 마카롱 아이스크림을 드립니다! [100명]
